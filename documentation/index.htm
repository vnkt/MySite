Template Installation

You can view this template in any web browser from your desktop computer. Because the files are written in HTML, you do not need to upload it to a server in order to display or edit the template.

Upload to The Server
  1. It's important to understand that you must have a server space to upload your template to. The server space can be purchased at many different places around the internet, if you're not experienced in this, search for 'web hosting' on Google.
  2. Select PerfectCV folder inside the same folder the documentation folder is. After that, you will see serveral folders and one index.html file. This is where all your CV contents will be. In addition to .html you should see a few files labeled .css inside the css folder. .css files are going to be the files that style the .html. It's very important to keep the .css files linked with in the HTML or the the design of this template will not look how it's meant to look. Along with the index.html and css folder, you will also have 5 additional folders to upload; bootstrap (this template is based on twitter bootstrap), fontello (icon fonts), img and php (phpmailer form files).
  3. Start your FTP client (We recommend Filezila or WinSCP which are free) and login to your server. You will need your server name (often times this will be your domain name), username, and password that you setup with the hosting company.
  4. Locate the domain folder you wish to upload your template files to. You are going to want to click and drag every file that you have labeled into that folder on your FTP client.
  5. Once the files are done uploading, you can try to access your website via browser.

HTML Structure

This CV template has a responsive layout with a set maximum width. This template uses the Twitter bootstrap framework which will help streamline any customization you wish to make. If you're unfamiliar with the Twitter Bootstrap fluid 12-column framework, be sure to read about it here.

The main structure of this site is wrapped in a 'container' div which is inside a 'wrapper' div. Inside of that div you will find each major section of the site given an appropriate fluid grid size all equalling 12 columns in length. For example, the '.aside_el' side section title element has a class of 'span3' while the contents next to it have a class of 'span9'. The total columns equals 12 columns.

	 <div class="container">
      <div class="row-fluid">
       <aside class="span3 aside_el"> ... </aside>
       <div class="span9"> ... </div>
      </div>
     </div>

If you would like to edit the color, font, or style of any elements in one of these columns, you would do the following in css/styles.css :

     .site_header h1 {
      /* Text */
      font: 6em 'Arvo', serif; /* 60px/10 */
      line-height: 0.66666666em; /* 40px/60px */
      font-weight: bold;
      color: #464646;
      /* Positioning & Display */
      display: inline-block;
      padding-right: 5px;
      padding-left: 5px;
      z-index: 2;
    }

If you find that your new style is not overriding, it is most likely because of a specificity problem. Scroll down in your CSS file and make sure that there isn't a similar style that has more weight.

So, to ensure that your new styles are applied, make sure that they carry enough "weight" and that there isn't a style lower in the CSS file that is being applied after yours and to the same element.

Portfolio/Gallery items

To change/insert an image into the portfolio section you need to upload them onto your server and change the img element src (source) location as well as href of the a element that links to the full size image. Also it's pretty easy to change the title and the category texts inside h2 and p elements below the image. Make sure you have a double size thumbnail in the same directory with @2x at the end of the file name so the image displays nicely on Retina screens (automatically via Retina.js plugin).

        <div class="galleryItem">
            <a href="img/2.jpg" class="lightbox['portfolio']"><img src="img/gallery_2.jpg" alt="" /></a>
            <h3>Business Design</h3>
            <p>Graphic Design</p>
        </div>

Contact form in the footer

The only thing that needs changing in the contact form is the email to deliver the messages to. You can edit this in the mailer.php file that is inside the php folder (You can do this with any code/text editor). You can change any other settings in the file if you want, but the email address is located at the following line:

        $address = "mail@gmail.com"; // YOUR EMAIL ADDRESS

CSS Structure

I'm using one main CSS file (css/layout.css), in this template which you are free to modify depending on the customization you require. There are also two additional style sheets that I do not recommend changing. Those style sheets reside in the 'css' folder. The six additional style sheets are:

  • styles.css - The main stylesheet you are encouraged to modify.
  • bootstrap.css - This is a main stylesheet that required for basic template styling like form, input, div, list element and etc. and it's located inside the bootstrap folder
  • bootstrap-responsive.css - This stylesheet use for responsively the theme for other devices screen size support. It is located inside the bootstrap folder as well.

The main CSS file contains all of the specific stylings for the template. It is located in the 'css' folder and it's named styles.css. There's two other css files in that folder, one for skill bar and one for the lightbox plugin. The file is separated into the following sections:

    
    /*

    1.  GLOBAL

    2.  BODY
    3.  NAVIGATION
    4.  HEADER
    5.  ABOUT ME
    6.  EXPERIENCE
    7.  EDUCATION
    8.  PORTFOLIO
    9.  FOOTER

    10.  RESPONSIVE

    */

If you would like to edit a specific section of the site, simply find the appropriate label in the file, and then scroll down until you find the appropriate class that needs to be edited.
